## Tech Requirements
- Node 22
- Prisma 7
- Postgresql

## Manual Setup
- Create database manually

## Getting Started
- Install Dependencies
```bash
npm install
```
- Run database scripts or prisma
```bash
npx prisma migrate dev --name init
npx prisma generate
```
Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
